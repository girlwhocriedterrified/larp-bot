# LARP Wagon Discord Bot

A custom Discord bot for the LARP Wagon Minecraft server.

## Commands

- `/mcupdate` - Update Minecraft server status (owner only)
- `/boot @user` - Boot user from voice chat
- `/mimic @user message` - Send message as another user (owner only)
- `/poll question option1 option2 duration` - Create an interactive poll

## Setup

1. Clone the repository
2. Run `npm install`
3. Set environment variables
4. Run `npm start`
